#  Ejercicios  de Programación Web

**Ejercicio 1:**  En este primer ejercicio instalamos XAMPP para prender el servidor local (Apache) en la computadora. Después, creamos la carpeta principal llamada `prograweb` dentro de `htdocs` junto con sus subcarpetas para tener ordenados los estilos, códigos y fotos (`css`, `js`, `img`). 

Al final, abrimos todo en Visual Studio Code para hacer una página HTML súper básica usando el atajo `!`. El objetivo fue lograr que el navegador web abriera nuestra página en el enlace `localhost` y mostrara en pantalla el mensaje "¡Hola Mundo! Esta es mi primer ejercicio de HTML." sin ningún error.

* **Ejercicio 2:** Aquí practicamos cómo separar textos usando párrafos y saltos de línea. También aprendimos a resaltar palabras importantes poniéndolas en negritas o cursivas directamente desde el código.
* **Ejercicio 3:** En esta práctica aprendimos a conectar páginas. Pusimos enlaces para ir a sitios externos como Google o Facebook abriéndolos en pestañas nuevas, y también un enlace para regresar al Ejercicio 1.
* **Ejercicio 4:** Aquí jugamos con imágenes. Aprendimos a meter una foto guardada en nuestra carpeta del proyecto, ajustarle el tamaño para que no se viera gigante y ponerle un texto de ayuda por si la imagen no cargaba bien.
* **Ejercicio 5:** Juntamos todo lo anterior en un solo lugar. Armamos una especie de "Blog Personal" combinando párrafos, metiendo imágenes flotantes a la mitad del texto y añadiendo botones con enlaces directos.
* **Ejercicio 6:** En este último ejercicio empezamos a darle orden profesional a la web. Dividimos la página en zonas usando las secciones oficiales: una cabecera para el título, un bloque central para el contenido fuerte y un pie de página para los derechos de autor.

* **Ejercicio 7:** En esta práctica armamos una página de una receta de Tacos al Pastor. El chiste aquí fue aprender a usar listas. Combinamos listas con viñetas para poner los ingredientes (metiendo una lista adentro de otra para detallar cómo iba cortada la piña) y listas numeradas para detallar el paso a paso de la preparación en orden.
* **Ejercicio 8:** Aquí diseñamos un horario semanal de clases de Ingeniería en Sistemas. Fue la primera vez que separamos el código conectando un archivo CSS externo (`ejercicio8.css`). Aprendimos a estructurar datos en filas y columnas usando tablas, y practicamos trucos avanzados para fusionar celdas (hacer que una materia abarcara dos horas o dos días seguidos) además de aplicarle colores llamativos a los laboratorios.

* **Ejercicio 9:** En este ejercicio armamos un formulario de contacto bastante completo. Lo interesante aquí fue que usamos etiquetas especiales de HTML5 para validar de forma automática que la gente escriba correos electrónicos reales o no deje campos obligatorios vacíos. Además, le conectamos un archivo CSS externo (`ejercicio9.css`) usando variables de colores y un diseño de columnas ordenadas para que visualmente pareciera un formulario profesional, cambiando de color los campos cuando haces clic sobre ellos.

* **Ejercicio 10:** En esta práctica creamos una calculadora para convertir grados Celsius a Fahrenheit. Fue el primer ejercicio donde metimos lógica con JavaScript (`EJERCICIO_10.js`) para capturar el número que escribe el usuario, hacer la operación matemática en el fondo y mostrar el resultado al instante sin recargar la página. Además, bloqueamos el campo del resultado con la propiedad `readonly` para que nadie pueda modificar el número final a mano, y le añadimos alertas que avisan si intentas picarle al botón dejando el espacio vacío o escribiendo letras.
